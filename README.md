### Commands to make this work

```bash
# If you don't have a python env
python -m venv iot

# Activate the env
source ./iot/bin/activate # on linux
source ./iot/scripts/activate # on windows
```

```bash
# Install all dependencies
pip install -r requirements.txt

# Start the module
python ./start.py
```
